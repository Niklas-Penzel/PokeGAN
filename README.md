# PokeGAN

## Usage:
* start scraping pokemon images with 
```
python scraper.py
```

## needed Packages:
* tqdm 
```
conda install tqdm
```
or
```
pip install tqdm
```
* numpy 
```
conda install numpy
```
or
```
pip install numpy
```
* opencv
```
conda install opencv
```
or
```
pip install opencv
```
